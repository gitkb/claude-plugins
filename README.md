# GitKB Claude Plugins

A [Claude Code plugin marketplace](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/plugins) by GitKB.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [meta](https://github.com/gitkb/meta) | Multi-repo workspace management — skills, hooks, and MCP server. |
| [gitkb](https://github.com/gitkb/gitkb-claude-plugin) | Knowledge base and code intelligence for AI-native development. |

## Usage

```bash
# Add this marketplace
claude plugin marketplace add gitkb/claude-plugins

# Install a plugin
claude plugin install meta@gitkb
claude plugin install gitkb@gitkb
```
