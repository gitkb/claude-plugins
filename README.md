# Harmony Labs Claude Plugins

A [Claude Code plugin marketplace](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/plugins) by Harmony Labs.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [meta](https://github.com/gitkb/meta) | Multi-repo workspace management — skills, hooks, and MCP server. |

## Usage

```bash
# Add this marketplace
claude plugin marketplace add gitkb/claude-plugins

# Install a plugin
claude plugin install meta@harmony-labs
```
